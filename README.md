# Le Poulailler

Application de gestion d'élevage avicole (React + Vite + PWA).

**Créateur :** Seydou Bakhayokho — [sdbsntech@gmail.com](mailto:sdbsntech@gmail.com) pour toute information ou suggestion.

## Fonctionnalités

- Lots, phases automatiques (J+15, J+30, J+45), ventes et décès
- Totaux en direct, alimentation & eau calculées
- Alertes médicaments et repas (notifications sur l'appareil)
- **Conseils essentiels** : gestion des poussins, vente, emplacement et construction du poulailler

## Lancer

```bash
npm install
npm run dev
```

## Build & PWA

```bash
npm run build
npm run preview
```

Données sauvegardées dans le navigateur (localStorage).
